# Neptune Product

Production starter for Neptune, a login-gated maritime intelligence product with public marketing pages and an internal admin console.

## Included

- Landing page
- Lead magnet capture
- Pricing page
- Login-gated admin dashboard
- CRM dashboard
- Subscription dashboard
- Website traffic dashboard
- Demo cookie login

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.
