# Neptune Product Roadmap

## Phase 1: Commercial foundation

- Public landing page
- Lead magnet capture
- Pricing page
- Login-gated admin dashboard
- CRM dashboard
- Subscription dashboard
- Website traffic dashboard

## Phase 2: Production backend

- Replace demo cookie auth with Supabase/Auth.js
- Add PostgreSQL schema for users, organizations, leads, accounts, subscriptions, page views, and events
- Add role-based access control
- Store leads in database instead of in-memory demo store

## Phase 3: Billing and analytics

- Stripe products, prices, checkout, and customer portal
- Webhook handling for subscription status
- Analytics ingestion for page views, conversions, campaigns, and source tracking

## Phase 4: Maritime operating modules

- Fleet and vessel records
- Voyages and port calls
- Safety workflows
- PMS, inventory, procurement, and documents
- Executive analytics and Neptune AI
